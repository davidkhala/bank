# Relationship Manager (RM)
企业客户经理
- 负责拉生意、维护客户、推动贷款审批的人。
# facility
facility = 授信/贷款安排。
- 银行给客户的proposal
- RM 要向客户解释银行打算怎么借钱给你、借多少、借多久、利率多少、要什么抵押品、有什么条件。
- aka. facility structure, facility letter


# Financial Covenants
銀行在貸款協議中設定的財務指標要求，用來確保借款人的財務狀況維持在可接受的風險水平。

## Leverage / Gearing
控制負債水平
- e.g. Debt / Equity ≤ 1.5x

## Interest Coverage Ratio (ICR)
確保有能力支付利息
- e.g. EBITDA / Interest ≥ 3.0x

## Debt Service Coverage Ratio (DSCR)
確保有能力償還本息
- e.g. Cashflow / Debt Service ≥ 1.2x

## Current Ratio / Quick Ratio
短期償債能力

## Net Worth Covenant
維持最低淨值

## Material Adverse Change (MAC)
不得出現重大不利變化
- 銀行主觀判斷

# Event of Default (EoD)
貸款協議中列明的、只要發生就會讓銀行有權採取行動的「重大違約情況」。
